Servicios brindados

Proveemos un servicio avanzado de información en base de datos en la nube de mascotas, la veterinaria o revendedor de este sistema crea un código QR que a su vez impacta en una base de datos donde quedan almacenados los datos del animal, si alguna vez el mismo se perdiera y lleva el código QR en alguna parte como una chapa identificatoria o pegado en el collar, sería muy fácil acceder a la información desde cualquier dispositivo móvil No es necesario realizar ninguna operación invasiva ni similarsólo imprimir el código en algun sticker para que puedan ser cargados por los animales a todos lados.
